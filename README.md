# ToMiddleChinese

Install:

```sh
$ pip install ToMiddleChinese
```

Usage:

```python
>>> import ToMiddleChinese
>>> ToMiddleChinese.get_unt('一天')
'一(ʔit˥)天(tʰen˦)'
```

- `ToMiddleChinese.get_qimyonhmieuzsjyt_list`
